Данный репозиторий это единый сервис авторизации mID.

Запуск сервиса:
Перед запуском необходимо заполнить файл `.env.example` в корне проекта,
указав необходимые переменные окружения, и переименовать его в `.env`.

После этого можно запустить сервис с помощью Docker Compose:

```bash
docker-compose up -d --build
```
